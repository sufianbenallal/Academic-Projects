# C Program and Makefile Build System

Basic systems programming project demonstrating how to compile and execute C programs using a Makefile in a Linux environment.

## Overview

This project introduces the fundamentals of C programming and build automation using Makefiles, a key tool in systems programming and software development.

## Features

- Simple C program (`hello.c`, `test.c`)
- Compilation into executable binaries
- Automated build process using Makefile
- Execution of compiled programs

## Makefile

The Makefile automates:

- Compilation of source files
- Linking into executable
- Clean-up of generated files

Example commands:

```bash
make        # compile program
make clean  # remove binaries
