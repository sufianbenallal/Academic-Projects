# Smart Room Web Application (Advanced)

Enhanced version of a smart room simulation web application with improved validation, interactivity and client-side data persistence.

## Features

- Multi-page web application:
  - `index.html` → Main dashboard :contentReference[oaicite:0]{index=0}
  - `config.html` → Room and sensor configuration :contentReference[oaicite:1]{index=1}
  - `minombre.html` → User input and validation :contentReference[oaicite:2]{index=2}

- User data storage using `localStorage`
- Dynamic welcome message based on stored user name
- Sensor activation system using checkboxes and persistent state
- Data transfer between pages via URL parameters
- Google Maps link generation from GPS coordinates
- Password validation with security checks (minimum length)
- Country validation with predefined allowed values
- Temperature validation with dynamic feedback and constraints
- Email validation:
  - Basic validation
  - Advanced validation using Regular Expressions (RegEx)

## Concepts Covered

- HTML5 layouts and forms
- CSS styling and UI structuring

- Local Storage (persistent client-side data)
- Form validation and user input handling
- URL parameter handling (GET requests)
- Regular expressions (RegEx)
- Event-driven programming
