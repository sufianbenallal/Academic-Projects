# Smart Room Web Application

Web application that simulates a smart room environment with configurable parameters and sensor visualization.

## Features

- Multi-page web application (index, configuration, user input)
- Dynamic layout using HTML and CSS
- Form-based configuration system
- Sensor selection and visualization
- Local storage using browser `localStorage`
- Dynamic welcome message based on stored user data
- Data transfer between pages using URL parameters
- Input validation and password checking
