# Academic-Projects
Portfolio of academic projects in networking, telecommunications, Linux systems, cloud and software development (Python, C, AWS).

## Projects

### Python Network Chat
Client-server and hybrid P2P chat application using Python sockets and select.

### Asterisk PBX VoIP
VoIP system configuration with Asterisk, IVR, voicemail, queues and inter-PBX connectivity.

### Linux Routing with FRRouting
Static routing, OSPF, IPv6-over-IPv4 tunnels and RIPng configuration in Linux virtual machines.

### AWS Cloud Deployment
Deployment of web services on AWS EC2 and integration with API Gateway and Lambda.

### Operating Systems C Labs
C programming projects using processes, signals, pipes, FIFOs, mmap and POSIX system calls.
