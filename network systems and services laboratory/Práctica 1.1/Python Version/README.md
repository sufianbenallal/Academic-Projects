## Python Implementation (High-Level)

This version of the project is developed in Python using the built-in socket library, providing a higher-level abstraction of network communication compared to C.

### Features

- TCP client/server implementation
- UDP client/server implementation
- Simplified socket handling using Python APIs
- RTT (Round Trip Time) measurement with statistical analysis
- Support for both IPv4 and IPv6
- Timeout handling and exception management

### Files

- `TCP_cliente.py` → TCP client
- `TCP_servidor.py` → TCP server
- `UDP_cliente.py` → UDP client
- `UDP_servidor.py` → UDP server

### Concepts Covered

- Socket programming in Python
- TCP vs UDP communication
- Client-server architecture
- RTT calculation and statistics
- Exception handling and timeouts
- High-level abstraction of networking
