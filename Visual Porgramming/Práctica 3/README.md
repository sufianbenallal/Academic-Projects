# Advanced Desktop Clock Application (C#)

Windows desktop application that combines analog and digital clocks with advanced features such as time zone management and user interaction.

## Features

- Real-time analog clock visualization
- Digital clock display
- Time editing functionality
- Time zone configuration and management
- Multiple interactive windows (dialogs and forms)
- Event-driven updates using timers
- Modular application design

## Project Structure

- `Program.cs` → Application entry point
- `RelojAnalogico.cs` → Analog clock rendering
- `RelojDigital.cs` → Digital clock logic
- `DlgDatosZona.cs` → Time zone configuration dialog
- `EditHora.cs` → Time editing interface
- `Acercade.cs` → About window

## Concepts Covered

- Object-Oriented Programming (OOP)
- Event-driven programming
- GUI development with Windows Forms
- Dialog and multi-form interaction
- Time management and formatting
- Graphics rendering in C#
- Modular architecture

## Technologies

- C#
- .NET Framework
- Windows Forms
