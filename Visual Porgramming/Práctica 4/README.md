# World Clock Desktop Application (C#)

Advanced Windows desktop application that manages multiple time zones with both analog and digital clock visualization.

## Features

- Real-time analog and digital clock display
- Time zone management system
- Add, edit and delete time zones (CRUD operations)
- Multi-window interface with dialogs
- Custom time editing functionality
- Event-driven updates using timers
- Modular and scalable architecture

## Project Structure

- `Program.cs` → Application entry point
- `RelojAnalogico.cs` → Analog clock rendering
- `RelojDigital.cs` → Digital clock logic
- `ZonaHoraria.cs` → Time zone data model and logic
- `DlgDatosZona.cs` → Add/Edit time zone dialog
- `DlgEliminarZona.cs` → Delete time zone dialog
- `EditHora.cs` → Time editing interface
- `Acercade.cs` → About window

## Concepts Covered

- Object-Oriented Programming (OOP)
- GUI development with Windows Forms
- Multi-window and dialog interaction
- CRUD operations in desktop applications
- Event-driven programming
- Time zone management logic
- Modular architecture and code organization
