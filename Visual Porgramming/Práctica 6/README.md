# Interactive Desktop Application (C#)

Windows desktop application focused on user interaction and dynamic interface behavior.

## Features

- Interactive graphical user interface
- Event-driven user interaction
- Dynamic UI updates based on user actions
- Form-based application logic
- Modular application structure

## Project Structure

- `Program.cs` → Application entry point
- `Form1.cs` → Main interface and application logic

## Concepts Covered

- Object-Oriented Programming (OOP)
- Event-driven programming
- GUI development with Windows Forms
- User interaction handling
- Application logic design

## Technologies

- C#
- .NET Framework
- Windows Forms
