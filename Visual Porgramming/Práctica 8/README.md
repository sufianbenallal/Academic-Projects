# ASP.NET Web Application with CRUD Functionality

Full-featured web application developed using ASP.NET Web Forms and C#, implementing dynamic pages and CRUD operations.

## Features

- Multi-page web application
- CRUD operations:
  - Add records (`añadir.aspx`)
  - Delete records (`borrar.aspx`)
- Server-side logic using C#
- Master pages for consistent layout
- Navigation between pages
- Dynamic content rendering

## Project Structure

- `Default.aspx.cs` → Main page logic
- `About.aspx.cs` → About page
- `Contact.aspx.cs` → Contact page
- `añadir.aspx.cs` → Add new data
- `borrar.aspx.cs` → Delete data
- `Global.asax.cs` → Application lifecycle management
- `Site.Master.cs` → Main layout
- `Site.Mobile.Master.cs` → Mobile layout
- `ViewSwitcher.ascx` → View switching component

## Concepts Covered

- ASP.NET Web Forms
- Server-side programming in C#
- CRUD operations in web applications
- Page lifecycle management
- Master pages and layout design
- Modular web architecture

- C#
