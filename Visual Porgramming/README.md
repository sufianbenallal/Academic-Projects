# Desktop Clock Application (C#)

Windows desktop application that displays a real-time analog clock with graphical rendering and interactive interface.

## Features

- Real-time analog clock visualization
- Custom graphical rendering
- Windows Forms interface
- Multiple forms (main window and additional screens)
- Event-driven updates
- User interface components and navigation

## Project Structure

- `Program.cs` → Application entry point
- `Form1.cs` → Main interface and logic
- `RelojAnalogico.cs` → Clock rendering and logic
- `Acercade.cs` → About window

## Concepts Covered

- Object-Oriented Programming (OOP)
- Event-driven programming
- GUI development with Windows Forms
- Graphics and drawing in C#
- Application structure in .NET

## Technologies

- C#
- .NET Framework
- Windows Forms
