# C# Database Application with SQL Integration

Desktop application developed in C# that integrates a relational database and applies a layered architecture using a Business Logic Layer (BLL).

## Features

- Database creation and management using SQL scripts
- Data insertion and querying
- Business Logic Layer (BLL) implementation
- Structured data processing
- Separation between application logic and data layer

## Project Structure

- `CFamosoBLL.cs` → Business Logic Layer implementation
- `dbo.famoso.sql` → Database schema and structure
- `Practica-7_script_bd.sql` → SQL scripts for database setup

## Concepts Covered

- Database design (SQL)
- Data persistence
- Layered architecture (BLL)
- Separation of concerns
- Backend logic in C#
- Interaction between application and database

## Technologies

- C#
- .NET Framework
- SQL (MySQL / SQL Server depending on config)
