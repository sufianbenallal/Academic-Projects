# Multi-Window Desktop Application (C#)

Windows desktop application that demonstrates multi-form interaction using parent and child windows.

## Features

- Multi-window interface (main form + child windows)
- Parent-child form communication
- Event-driven UI interaction
- Navigation between different application views
- Modular Windows Forms design

## Project Structure

- `Program.cs` → Application entry point
- `Form1.cs` → Main window (parent form)
- `VentanaHija.cs` → Child window
- `AboutBox1.cs` → About dialog

## Concepts Covered

- Object-Oriented Programming (OOP)
- Event-driven programming
- GUI development with Windows Forms
- Multi-form communication
- Application navigation structure

## Technologies

- C#
- .NET Framework
- Windows Forms
