# Time Management System in C++

C++ project that implements time handling and manipulation using an object-oriented approach.

## Features

- Custom class for time representation (hours, minutes, seconds)
- Time manipulation and formatting
- Modular structure using `.cpp` and `.h` files
- Utility functions for data processing
- Dynamic memory management support

## Project Structure

- `Practica3.cpp` → Main program
- `CHora.cpp` → Time class implementation
- `CHora.h` → Time class definition
- `Utils.cpp` → Helper functions
- `MemoryManager.h` → Memory management utilities

## Concepts Covered

- Object-Oriented Programming (OOP)
- Class design and encapsulation
- Modular programming in C++
- Code reuse and abstraction
- Time data handling and formatting

## Technologies

- C++
- Standard Library
