# Record and Entity Management System in C++

C++ application that models a complete system with multiple interacting entities, using object-oriented programming and modular design.

## Features

- Multiple entity classes:
  - `cliente` → Client management
  - `empleado` → Employee management
  - `registro` → Record handling
  - `CFicha` → Data structure representation
  - `CHora` → Time management
- Interaction between multiple objects
- Modular architecture (`.cpp` + `.h`)
- Data modeling and structured design
- Utility functions for processing
- Memory management support

## Project Structure

- `Practica7-2.cpp` → Main program
- `cliente.cpp / cliente.h` → Client entity
- `empleado.cpp / empleado.h` → Employee entity
- `registro.cpp / registro.h` → Record system
- `CFicha.cpp / CFicha.h` → Data structure
- `CHora.cpp / CHora.h` → Time handling
- `utils.cpp / utils.h` → Helper functions
- `MemoryManager.h` → Memory utilities

## Concepts Covered

- Advanced Object-Oriented Programming (OOP)
- Data modeling with multiple entities
- Class relationships and interaction
- Modular programming in C++
- System design and architecture
- Code organization and scalability
