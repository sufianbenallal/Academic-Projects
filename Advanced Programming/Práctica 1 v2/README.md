# Advanced Memory Management and Modular Programming in C++

Improved version of a C++ project focused on modular programming and memory management.

## Improvements Over Previous Version

- Better code organization and structure
- Improved modular design using header and source files
- Enhanced memory management handling
- Cleaner separation between logic and declarations
- More maintainable and scalable codebase

## Features

- Modular program structure (`.cpp` + `.h`)
- Custom function implementations
- Memory management utilities
- Structured and reusable code design

## Project Structure

- `practica1.cpp` → Main program
- `funciones.cpp` → Function implementations
- `funciones.h` → Function declarations
- `MemoryManager.h` → Memory management utilities
