# Advanced Time Management System v2 in C++

Third iteration of a C++ time management project, focusing on improving architecture, code quality and maintainability.

## Improvements Over Previous Versions

- Further refined class design and encapsulation
- Improved modular architecture and code organization
- Cleaner and more readable implementation
- Enhanced utility functions
- Better handling of time-related operations
- Increased robustness and maintainability

## Features

- Custom class for time representation (hours, minutes, seconds)
- Time manipulation and formatting
- Modular program structure (`.cpp` and `.h`)
- Utility functions for data processing
- Memory management support

## Project Structure

- `Practica4.cpp` → Main program
- `CHora.cpp` → Time class implementation
- `CHora.h` → Time class definition
- `Utils.cpp` → Helper functions
- `Utils.h` → Utility declarations
- `MemoryManager.h` → Memory management utilities

## Concepts Covered

- Advanced Object-Oriented Programming (OOP)
- Code refactoring and iterative improvement
- Modular programming in C++
- Encapsulation and abstraction
- Software maintainability and scalability

## Technologies

- C++
- Standard Library
