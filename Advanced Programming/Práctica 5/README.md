# Object-Oriented Record Management System in C++

C++ application that demonstrates advanced object-oriented programming concepts by combining multiple classes into a structured system.

## Features

- Custom class for time management (`CHora`)
- Record/entity class (`CFicha`) for structured data handling
- Interaction between multiple classes (composition)
- Modular architecture with `.cpp` and `.h` files
- Utility functions for data processing
- Memory management support

## Project Structure

- `Practica5.cpp` → Main program
- `CFicha.cpp` → Record class implementation
- `CFicha.h` → Record class definition
- `CHora.cpp` → Time class implementation
- `CHora.h` → Time class definition
- `Utils.cpp` → Helper functions
- `Utils.h` → Utility declarations
- `MemoryManager.h` → Memory utilities

## Concepts Covered

- Object-Oriented Programming (OOP)
- Class composition (objects inside objects)
- Encapsulation and abstraction
- Modular programming in C++
- Data modeling and structured design

## Technologies

- C++
- Standard Library
