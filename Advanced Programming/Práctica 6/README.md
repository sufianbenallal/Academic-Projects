# Polynomial Management System in C++

C++ application that models and manipulates polynomials using object-oriented programming and modular architecture.

## Features

- Custom classes for mathematical structures:
  - `CMonomio` → Monomial representation
  - `CTermino` → Polynomial term handling
  - `CPolinomio` → Polynomial operations
- Mathematical modeling using objects
- Modular architecture with multiple source and header files
- Dynamic memory management
- Utility functions for processing

## Project Structure

- `Practica6-v1.cpp` → Main program
- `CMonomio.cpp` → Monomial implementation
- `CMonomio.h` → Monomial definition
- `CTermino.cpp` → Term implementation
- `CTermino.h` → Term definition
- `CPolinomio.cpp` → Polynomial logic
- `CPolinomio.h` → Polynomial definition
- `MemoryManager.h` → Memory utilities

## Concepts Covered

- Advanced Object-Oriented Programming (OOP)
- Class relationships and abstraction
- Mathematical modeling with software
- Modular programming in C++
- Code organization and scalability
