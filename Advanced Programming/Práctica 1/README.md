# Memory Management and Modular Programming in C++

C++ project that demonstrates modular programming and memory management techniques using multiple source and header files.

## Features

- Modular program structure using `.cpp` and `.h` files
- Custom function implementation
- Memory management handling
- Separation between logic and declarations
- Structured program organization

## Project Structure

- `Practica1.cpp` → Main program
- `Funciones.cpp` → Function implementations
- `funciones.h` → Function declarations
- `MemoryManager.h` → Memory management utilities

## Concepts Covered

- Modular programming in C++
- Header and source file separation
- Memory management concepts
- Function abstraction
- Code organization

## Technologies

- C++
- Standard Library
