# Matrix Operations Library in C++

C++ project that implements matrix operations using an object-oriented approach and modular architecture.

## Features

- Custom matrix class for floating-point values
- Dynamic memory management for matrix storage
- Modular structure using multiple source and header files
- Utility functions for matrix operations
- Encapsulation of data and operations

## Project Structure

- `Practica2.cpp` → Main program
- `CMatFloat.cpp` → Matrix class implementation
- `CMatFloat.h` → Matrix class definition
- `utils.cpp` → Helper functions
- `utils.h` → Utility declarations
- `MemoryManager.h` → Memory management utilities

## Concepts Covered

- Object-Oriented Programming (OOP)
- Classes and encapsulation
- Dynamic memory management
- Modular programming in C++
- Abstraction and code reuse

## Technologies

- C++
- Standard Library
