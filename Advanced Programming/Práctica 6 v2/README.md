# Advanced Polynomial Management System in C++

Improved version of a polynomial management system implemented in C++ using object-oriented programming.

## Improvements Over Previous Version

- Refined class design for better abstraction
- Improved interaction between polynomial and monomial classes
- Cleaner and more maintainable code structure
- Enhanced modular architecture
- Optimized implementation of mathematical operations
- Better scalability and extensibility

## Features

- Object-oriented modeling of polynomials
- Classes for:
  - `CMonomio` → Monomial representation
  - `CPolinomio` → Polynomial operations
- Modular architecture (`.cpp` and `.h`)
- Dynamic memory management
- Mathematical computation handling

## Project Structure

- `Practica6-v2.cpp` → Main program
- `CMonomio.cpp` → Monomial implementation
- `CMonomio.h` → Monomial definition
- `CPolinomio.cpp` → Polynomial logic
- `CPolinomio.h` → Polynomial definition
- `MemoryManager.h` → Memory utilities

## Concepts Covered

- Advanced Object-Oriented Programming (OOP)
- Code refactoring and optimization
- Abstraction and encapsulation
- Modular programming in C++
- Mathematical modeling and system design

## Technologies
