# Advanced Time Management System in C++

Improved version of a C++ time management project using object-oriented programming and modular architecture.

## Improvements Over Previous Version

- Refactored class design for better encapsulation
- Improved modular structure (`.cpp` and `.h`)
- Cleaner and more maintainable codebase
- Enhanced utility functions
- Better separation of responsibilities
- More robust time handling logic

## Features

- Custom class for time representation (hours, minutes, seconds)
- Time manipulation and formatting
- Modular architecture
- Utility functions for processing
- Memory management support

## Project Structure

- `Practica3.cpp` → Main program
- `CHora.cpp` → Time class implementation
- `CHora.h` → Time class definition
- `Utils.cpp` → Helper functions
- `Utils.h` → Utility declarations
- `MemoryManager.h` → Memory management utilities

## Concepts Covered

- Advanced Object-Oriented Programming (OOP)
- Code refactoring and improvement
- Encapsulation and abstraction
- Modular programming in C++
- Software maintainability

## Technologies

- C++
- Standard Library
