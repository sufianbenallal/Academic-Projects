# Statistical Data Processing System in C

Modular C application designed to process and analyse data using basic statistical operations.

## Features

- Data input handling
- Statistical processing (mean, calculations, etc.)
- Output generation
- Modular architecture with multiple source and header files

## Project Structure

- `main.c` → Main program flow
- `entrada.c` → Input data handling
- `salida.c` → Output processing
- `Estadistica.c` → Statistical functions implementation
- `Estadistica.h` → Function declarations and structures

## Concepts Covered

- Modular programming in C
- Header files and compilation units
- Separation of concerns
- Data processing and analysis
- Functions and structured code design

## Technologies

- C
- Standard Input/Output
- Multi-file project structure
